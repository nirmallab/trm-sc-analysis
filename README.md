# Installation
Package management is done with [pixi](https://pixi.sh/latest/). To install run:
```
pixi install
```
Use the environment with `pixi shell`. 

# About
These are notebooks for processing single-cell RNA sequencing data from experiments BRI-2937/39/41. Data was processed according to the [single-cell best practices](https://www.sc-best-practices.org/preamble.html) with scanpy.
